#  Demoblaze Automation Framework

##  Project Overview

This project is a Selenium + TestNG automation framework built for testing the Demoblaze e-commerce application.

##  Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* Extent Reports

##  Project Structure

* `src/` → Test scripts and Page Object Model
* `test-data/` → Excel test data
* `test-deliverables/` → QA documents (Test Plan, Reports)
* `screenshots/` → Failure screenshots
* `reports/` → Automation reports

## How to Run Tests

1. Clone the repository
2. Open in Eclipse/IDE
3. Run `testng.xml`

## Reports

* Extent Report → `reports/`
* Screenshots → `screenshots/`
